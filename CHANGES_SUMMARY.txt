                          DESIGN SYSTEM DETAILS
═══════════════════════════════════════════════════════════════════════════════

COLOR PALETTE
─────────────
Primary:    Blue-600 → Cyan-600 (gradients)
Success:    Emerald-500 (real news, positive)
Danger:     Red-500 (fake news, alerts)
Warning:    Amber-500 (neutral, tips)
Neutral:    Slate-50 to Slate-950 (all backgrounds/text)
Backdrop:   White/Black with 70% opacity

TYPOGRAPHY
──────────
Headings:   Bold, 2xl-4xl, gradient or solid colors
Body:       Regular, slate-600 light / slate-300 dark
Labels:     Small, semibold, muted slate colors
Font-family: System fonts (Segoe UI, Roboto, etc.)

EFFECTS & ANIMATIONS
────────────────────
Backdrop Blur:  16px (glassmorphism)
Borders:        1-2px with transparency
Shadows:        lg and xl on hover
Transitions:    200-300ms smooth
Animations:     spin (loading), bounce (icons)
Transforms:     scale on hover, width on load

SPACING
───────
Padding:  p-4, p-6, p-8 (consistent scale)
Margins:  mb-6, mb-8, mt-4 (breathing room)
Gaps:     gap-4, gap-6, gap-8 (column/row spacing)
Breakpoints: md: 768px, lg: 1024px

                          INSTALLATION STEPS
═══════════════════════════════════════════════════════════════════════════════

1. Install Dependencies:
   $ npm install
   
   This installs:
   - Tailwind CSS (v3.4.1)
   - PostCSS (v8.4.31)
   - Autoprefixer (v10.4.16)
   - All other project dependencies

2. Start Development Server:
   $ npm start
   
   The app will run on http://localhost:3000
   with hot reload enabled

3. Build for Production:
   $ npm run build
   
   Creates optimized build with CSS purged
   (only used styles included, ~15KB gzipped)
